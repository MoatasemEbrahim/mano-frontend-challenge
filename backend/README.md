# Backend Project - Node.js with TypeScript

This is the backend portion of the Mano Health Code Challenge, built using Node.js and TypeScript.

## Requirements

- **Node.js** (version 20 or higher)
- **pnpm** (version 8.4 or higher)

## Project Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MoatasemEbrahim/mano-frontend-challenge.git
   cd mano-frontend-challenge/backend
2. **Instal node modules:**
   ```bash
   pnpm install
   ```


## Run Locally

Start the server

```bash
pnpm dev
```
The server will start on port 8080 by default (http://localhost:8080)


## Build

To build this project run

```bash
pnpm build
```


## Serve built files

To run the project from the built files run (you must run build command before this command)

```bash
pnpm start
```
The server will start on port 8080 by default (http://localhost:8080)
