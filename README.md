# Mano Health Code Challenge

This repository contains the code solution for the Mano Health Company coding challenge. It was cloned from the initial repository provided for the task, which is available at [Mano AI Labs Frontend Challenge](https://github.com/mano-ai-labs/frontend-challenge-1).

## Repository Structure

The repository consists of two primary subdirectories (projects):
- **frontend**: The frontend project, built with React.js, Vite, and TypeScript.
- **backend**: The backend project, built with Node.js and TypeScript.

Each project has its own setup instructions and specific requirements detailed in their respective README files. For overall task requirements and guidelines, please refer to the original repository linked above.
